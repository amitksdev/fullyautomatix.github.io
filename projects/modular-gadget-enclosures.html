Modular Gadget Enclosures

A reusable enclosure library for hobby boards with matching inserts, vent patterns, and fastener geometry that simplifies polished gadget builds.

Build Summary

This synthetic collection includes swappable lids and internal trays so one chassis family can host sensor hubs, displays, or relay modules.

  • Template grid supports common 50x70mm and 85x56mm boards
  • Snap tabs or M3 insert options for final assembly
  • Heat-vent and cable gland cutouts generated per profile
  • Faceplate presets for OLED, rotary encoder, and USB-C

Technical Specs

CAD StackFusion 360 and FreeCAD compatible
MaterialPETG primary, TPU feet optional
Tolerance0.25mm clearance standard profile
Print Time2h to 8h depending on module size
Target CostUSD 8 to 14 filament and inserts
