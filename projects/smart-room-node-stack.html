Smart Room Node Stack

A room automation node with sensor layers, Home Assistant hooks, and a wall-mountable printed shell for fast deployment in every room.

Build Summary

This synthetic stack reports environmental data and triggers automations for climate, lighting, and occupancy-aware notifications.

  • Temperature, humidity, PIR, and VOC sensor modules
  • Printed backplate with hidden cable channel routing
  • Magnetic service panel for quick board replacement
  • MQTT auto-discovery profile for fast pairing
