Desk Signal Tower

A compact desktop status beacon with tactile mode switching and a snap-fit printed tower body for coding, streaming, or focus sessions.

Build Summary

The tower supports USB-C power and can map signal states to calendar events using a lightweight serial bridge.

  • Triple high-brightness diffused LEDs for visual clarity
  • Single-button short-press and long-press mode system
  • Printed lens diffuser stack with quarter-turn lock
  • Optional buzzer alert profile for timer events
